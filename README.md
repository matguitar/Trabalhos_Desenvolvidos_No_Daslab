# Resumo do projeto acadêmico

Este repositório conta com diversos relatórios de projetos de análise de dados realizados na iniciação científica da UFES via projeto DasLab. Esses 
trabalhos foram desenvolvidos em equipe (nomes dos integrantes nos relatórios). Foram realizadas diversas análises descritivas em pespectivas
espaciais e temporais. O foco foi analisar dados referentes ao uso de psicoativos no ES, focando em avaliar um projeto de lei implementados nos
municípios do chamado "Estado Presente", focando em verificar os comportamentos antes e depois da implementação do mesmo. O projeto contou com
diversas reuniões estratégicas e multidiciplinares com os professores reponsáveis (estatística e sociologia), além de profissionais que trabalhavam
em organizações públicas e disponibilizaram os dados para as análises.
